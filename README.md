# GA Convert Money

GA Convert Money is a simple currency converter project developed with HTML, CSS, and JavaScript. It allows you to convert values from Brazilian Real (BRL) to US Dollar (USD), Euro (EUR), and Bitcoin (BTC), using fixed exchange rates defined in the code.

## Features
- Convert values from Real to Dollar, Euro, or Bitcoin
- Intuitive and responsive interface
- Dynamic update of converted values
- Display of currency icons

## How to use
1. Enter the amount in Brazilian Real (BRL) in the input field
2. Select the currency to convert to (USD, EUR, or Bitcoin)
3. Click the "Convert" button
4. The converted value will be displayed below

## Technologies used
- HTML5
- CSS3
- JavaScript

## Project structure
```
index.html        # Main page
styles.css        # Interface styles
scripts.js        # Conversion logic
assets/           # Currency images
README.md         # Documentation
```

## Notes
- The exchange rates are fixed and may not reflect the real market value.
- To update the rates, edit the `scripts.js` file.
